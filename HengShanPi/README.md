# HengShanPi

## Hardware Specifications

- **Processor**: D133EBS
- **Memory**: 1MB SRAM + 8MB PSRAM
- **Storage**: 16 MB QSPI Nor Flash
- **Interfaces**: RGB565, LVDS, DVP Camera, Ethernet, PWM, I2C, SPI, QSPI, UART, GPIO
- **Power**: 5 ~ 12 V
