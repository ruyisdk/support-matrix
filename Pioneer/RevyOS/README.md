## Installation Steps

### Flashing Image

Decompress the image using `zstd`.
Write the image to the microSD card using `dd`.

```bash
zstd -d /path/to/revyos.img.zstd
dd if=/path/to/revyos.img of=/dev/yout-device bs=4M status=progress
```

### Common Issues
