## Installation Steps

### Flashing the Image to microSD Card Using `dd`

```shell
xzcat Fedora-38-Minimal-MilkV-Duo-riscv64.img.xz | sudo dd of=/dev/sdc bs=4M iflag=fullblock status=progress 
```
