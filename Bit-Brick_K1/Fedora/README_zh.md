### 硬件信息

- BIT-BRICK K1
- 电源适配器
- microSD 卡一张
- USB to UART 调试器一个

## 安装步骤

### 刷写镜像

使用 `gzip` 解压镜像。
使用 `dd` 将镜像写入 microSD 卡。

```bash
gzip -d Fedora-Minimal-42-20250324132559.riscv64.SpacemiT-K1.raw.gz
sudo dd if=Fedora-Minimal-42-20250324132559.riscv64.SpacemiT-K1.raw.gz of=/dev/<your_device> bs=4M status=progress
```

### 登录系统

通过串口登录系统。

默认用户名： `root`
默认密码： `riscv`
