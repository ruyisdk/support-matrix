## Installation Steps

### Flashing Image

Use `zstd` to decompress the image.
Use `dd` to flash the image to the microSD card.

```bash
zstd -d /path/to/fedora.raw.zst
sudo dd if=/path/to/fedora of=/dev/your-device bs=1M status=progress
```
