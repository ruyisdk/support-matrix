## Installation Steps

### Flashing the Image

Use `unxz` to decompress the image.
Use `dd` to flash the image to the microSD card.

```bash
unxz /path/to/ubuntu.img.xz
sudo dd if=/path/to/ubuntu.img of=/dev/your-device bs=1M status=progress
```
