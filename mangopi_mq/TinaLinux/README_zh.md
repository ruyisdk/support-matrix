## 安装步骤

### 烧写镜像

下载并解压后，使用 dd 将镜像烧写到 SD 卡：
```bash
sudo dd if=mq-r-f133-rtl8189fs-5113-dns-uart0.img of=/dev/your/device bs=1M status=progress
```

### 登录系统

通过串口登录系统。

TX, RX 分别为板子底部 (P3) 第7, 8号引脚。
